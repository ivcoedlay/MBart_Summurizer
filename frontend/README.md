AI Document Summarizer — Frontend

Этот проект представляет собой веб-интерфейс для загрузки документов и их автоматической суммаризации с помощью искусственного интеллекта.

---

Пользователи могут:

- Загружать документы в форматах `.docx`, `.odt`, `.txt` (до 15 МБ)
- Настраивать параметры суммаризации (мин./макс. длина результата)
- Просматривать результат обработки
- Отслеживать статус выполнения задачи
- Просматривать историю обработанных документов

Фронтенд взаимодействует с бэкендом через REST API по адресу `http://localhost:8000/api/v1`.

---

Технологии

- **React** + **TypeScript**
- **Vite** — сборка и dev-сервер
- **Tailwind CSS** — стилизация
- **React Query** — управление состоянием и кэширование запросов
- **Axios** — HTTP-клиент
- **React Dropzone** — drag & drop загрузка файлов
- **Lucide React** — иконки

Установка и запуск

1. Убедитесь, что установлены **Node.js ≥18** и **npm/yarn/pnpm**.
2. Установите зависимости:

```bash
npm install
```

3. Настройте переменные окружения (при необходимости):

```env
# .env
VITE_API_BASE_URL=http://localhost:8000/api/v1
```

4. Запустите локальный сервер разработки:

```bash
npm run dev
```

Приложение будет доступно по адресу: [http://localhost:5173](http://localhost:5173)

5. Для сборки production-версии:

```bash
npm run build
```

---

Ожидаемые эндпоинты (бекенд):

- `POST /documents/` — загрузка файла
- `GET /documents/` — список документов
- `GET /documents/{id}` — детали документа
- `POST /summaries/` — запуск суммаризации
- `GET /summaries/{id}` — статус и результат суммаризации

Бэкенд должен поддерживать CORS для `http://localhost:5173`.
