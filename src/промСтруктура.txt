# Суть проекта

Нейросеть - ИИ-помощник для создания саммари (краткого содержания) для загруженных файлов
    Поддерживаемые форматы: doc, docx, txt, odt
    Максимальный размер файла: 15 МБ
    Стек технологий: MongoDB, React/TypeScript (фронтенд), Python (бэкенд/нейросеть)
    Конкретные инструменты: PyCharm, MongoDB 8.2.1, MongoDB Compass
    Библиотеки: python-docx, React, FastAPI

На основе анализа приложенного проекта (gazeta) мы можем использовать:
    Предобученную модель mbart_ru_sum_gazeta для суммаризации русскоязычных текстов
    Опыт предобработки текста из проекта
    Архитектурные паттерны обработки текстовых данных
